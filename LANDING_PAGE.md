# Faculty Quest Landing Page

This directory contains the premium landing page for Faculty Quest app downloads.

## Structure
- `index.html` - Main landing page
- `styles.css` - Premium styling with animations
- `script.js` - Interactive features
- `downloads/` - APK and app files

## Deployment
Deploy this to Vercel separately from the main app:
```bash
cd landing-page
vercel
```

## Features
- Premium design with glassmorphism
- Animated hero section
- Download buttons for Android/iOS/Web
- FAQ section
- Responsive design
- SEO optimized
